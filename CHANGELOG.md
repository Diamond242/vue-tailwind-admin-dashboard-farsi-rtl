# Changelog

All notable changes to this project will be documented in this file.

## [2.0.2-fa.1] - 2025-12-30
### Added
- Full English/Persian localization with runtime language switch and persistence.
- RTL-first layout across navigation, menus, forms, tables, dropdowns, and notifications.
- Jalali (Shamsi) calendar view and localized date/time pickers.
- Persian typography (Vazirmatn) with easy font swapping.
- Locale-aware number and currency formatting (IRR for fa, USD for en).

### Changed
- Calendar toolbar layout for RTL consistency.
- Sidebar/header alignment and responsive RTL behavior.
- Form input icon placement and RTL spacing rules.
